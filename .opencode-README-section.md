## Working with OpenCode Agents

This project uses [OpenCode](https://opencode.ai) agents for AI-assisted development.

### Memory Bank

Project context is stored in `memory-bank/` for agent reference:
- **productContext.md**: Project overview and tech stack
- **activeContext.md**: Current focus and blockers  
- **systemPatterns.md**: Coding standards and architecture
- **decisionLog.md**: Architectural decisions with rationale
- **progress.md**: Task tracking (Done/Doing/Next)
- **taskRegistry.md**: Task dependencies and coordination

Update these files to maintain agent context across sessions.

### Task Registry

This project uses a task registry at `memory-bank/taskRegistry.md` to coordinate agent work:
- Tracks active work to prevent conflicts
- Coordinates multiple agents working on the same project
- Records completed tasks and resolutions

### Running Agents

To invoke OpenCode agents:
```bash
# Use OpenCode commands
/alspec    # Write technical specification
/alplan    # Write implementation plan
/al        # Align-and-Do (discuss + implement)
/code      # Direct code generation

# Or invoke specific agents
@frontend-specialist
@backend-specialist
@code-specialist
@documentation-writer
```

### Handovers

Agents document their work in handover files:
- Location: `handovers/{YYYYMMDD}_{TICKET}_{AGENT}.handover.md`
- Contains: Task summary, problems, recommendations, metadata
- For details: See `docs/Handover Template.md`
